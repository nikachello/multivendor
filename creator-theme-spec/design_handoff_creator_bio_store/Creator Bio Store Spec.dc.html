Design system spec

Creator Bio Store

A warm, editorial storefront kit that reads as one person's bio link, not a SaaS dashboard — terracotta accents and a serif display name give it the credibility of a real brand while staying quiet enough for any merchant's --primary to lead the page. Every surface sits a half-step off the page background, so cards feel placed rather than boxed, and dark mode is tuned separately — deep warm charcoal, not an inverted light theme — because bios get scrolled at night. It flexes from a drop-artist countdown to a WhatsApp order button without ever looking generic.
